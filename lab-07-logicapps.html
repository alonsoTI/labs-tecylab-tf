Laboratorio de Azure Logic Apps con Terraform

Este laboratorio te guiará para implementar una Azure Logic App que recibe datos de una solicitud HTTP y los almacena en Azure Blob Storage.

Prerrequisitos

  • Tener una cuenta de Azure.
  • Instalar Terraform.
  • Instalar Azure CLI.

Pasos

  1. Preparar el entorno: 
    mkdir logic-apps-terraform
    cd logic-apps-terraform
  2. Crear el archivo principal de Terraform: 
    touch main.tf
    nano main.tf

    Agrega el contenido que incluye la definición de la Logic App (ver sección anterior).

  3. Inicializar Terraform: 
    terraform init
  4. Planificar la implementación: 
    terraform plan
  5. Aplicar la configuración: 
    terraform apply

    Escribe yes para confirmar.

  6. Probar la Logic App:
    1. Después de implementar, Terraform mostrará la URL del desencadenador HTTP.
    2. Usa herramientas como Postman o curl para enviar una solicitud POST con el siguiente formato:
      3. 
curl -X POST -H "Content-Type: application/json" -d '{"name":"Juan"}' https://

      Reemplaza https://<URL-de-tu-logic-app> con la URL proporcionada por Terraform. La URL se encuentra en los valores de salida o en la pestaña de la Logic App en el portal de Azure.

  7. Verificar en Azure:

    Accede a tu cuenta de Azure Storage y verifica los blobs creados.

  8. Eliminar recursos: (opcional) 
    terraform destroy

    Escribe yes para confirmar.
